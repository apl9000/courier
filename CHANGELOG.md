# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-07

### Added
- Initial release of Courier email utility
- iCloud SMTP integration with app-specific password support
- Nodemailer integration for reliable email delivery
- Handlebars templating support for branded emails
- TypeScript type definitions
- Three pre-built email templates (welcome, notification, password-reset)
- Comprehensive documentation and examples
- Support for email attachments
- Multiple recipient support (to, cc, bcc)
- Named email addresses support
- SMTP connection verification
- MIT License

### Features
- Send simple text and HTML emails
- Template-based email sending
- Load templates from files
- Register inline templates
- Format email addresses with display names
- Full type safety with TypeScript
- Deno-native implementation
- JSR (JavaScript Registry) ready
